ReadWell Aid Test Document

This is a test document for the YusrBot chatbot feature.

Key Information:
- ReadWell Aid is a reading assistance application
- It features a chatbot that can answer questions about documents
- The chatbot uses OpenAI's Assistant API with vector stores for RAG
- Users can upload PDFs or paste text to chat about

Technical Features:
- File search capability for document understanding
- Speech-to-text input using Whisper API
- Text-to-speech output for responses
- Real-time chat interface with message history
